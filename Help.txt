Files:
The module code itself, is located at the 2nd kitten folder or at 'https://github.com/RobDaMob/Kitten/tree/master/Kitten'.
Commands:
1. The pip() command is used with the command pip('command','name','other'). All of the 3 parameters can be replaced with None. Same applies to all the other ones.
2. The close() command exits the program, no parameters needed. It basically does the exit() command.
3. The kill() command exits a process
